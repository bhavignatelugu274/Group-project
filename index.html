Weather Determination Project

A Group Research Project

Introduction

This project explores various methods of weather determination including traditional techniques, meteorological models, and AI-based forecasting. We aim to compare their accuracy and usability.

Methods

  • Data Collection from Weather APIs (OpenWeatherMap, WeatherStack)
  • Statistical Analysis & Pattern Recognition
  • Machine Learning Models for Forecasting
  • Comparison with Historical Data
